TeamUp
This application helps you find teammates for your specified events.
You can add events into waiting list, when there is a new user waiting for the same event,
you and all the users waiting for the same event will receive an email as a notification.
You can also search for the users waiting for the same event, and all the events of yours.
After set up your team, you can delete your waiting event from your list, or delete all your events.
